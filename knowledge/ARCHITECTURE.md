## 9. 关键配置参数

| 参数 | 冠军值 | 说明 |
|------|--------|------|
| `MAX_TOKENS` | 5000 | research/code 任务 token 上限 |
| `REVIEW_TOKENS` | 3000 | review 任务 token 上限 |
| `temperature` | 0.7 | 生成温度 |
| `MAX_RUNS` | 2 | MAX 策略运行次数 |
| `self_reflect` | research: ✓ / code: ✗ | 自反射开关 |
