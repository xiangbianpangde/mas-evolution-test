## 已识别的模式

### 1. 评估器分离模式
**描述**: Generator 不评自己，由独立 Evaluator 评分
**观察到**: OpenAI, Anthropic, 我们的实验
**原则**: Agent 总是高估自己的工作

### 2. 深度优先模式
**描述**: 大目标分解为小构建块，一次只做一个
**观察到**: OpenAI 官方经验
**原则**: 避免"一次搞定"导致的质量下降

### 3. 渐进式披露模式
**描述**: 先展示摘要，需要时再深入细节
**观察到**: 所有好的文档和报告
**原则**: 人类处理信息的方式

### 4. MAX 策略模式
**描述**: 运行多次取最优分数
**观察到**: v29-v33 实验
**原则**: 单次运行有方差，多次运行降低方差（但不是越多越好）

### 5. 黄金Token Budget 模式
**描述**: 存在一个最优 token 数量
**观察到**: v31 实验
**原则**: 太少不够深度，太多边际递减
