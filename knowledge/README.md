## 🔑 核心发现 (快速摘要)

### ✅ 有效策略
- **Core research**: 需要自反射
- **Token budget**: 5000 最佳
- **评估器**: code 从宽，research 从严

### ❌ 失败教训
- **Gen research**: 禁止自反射 (破坏结构)
- **Code (所有)**: 禁止自反射
- **超长 prompt**: 上下文窗口是稀缺资源
